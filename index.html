Getting Started

Welcome to the Stationery Management System! Here's how to get started:

  1. Login with your class or admin credentials
  2. View your class dashboard with student information
  3. Add new students using the "Add Student" button
  4. Track purchases and balances for each student
  5. Generate reports to analyze financial data
  6. Admins can manage class passwords and system settings

Admin Features

Admin Login: Use username "admin" and password "admin123"

Class Management: View and manage all classes

Password Management: Change passwords for any class

System Control: Full access to all student and financial data
